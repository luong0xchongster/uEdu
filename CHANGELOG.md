# Changelog

All notable changes to uEdu will be documented in this file.

## [1.0.0] - 2025-01-19

### Added
- English Academy Management System
- Go API with PostgreSQL
- Admin portal for managing students, teachers, courses, and exams
- Student portal with exam platform
- Support for three exam types:
  - Pre-registration exams (placement tests)
  - Progress exams (during study)
  - Final exams (course completion)
- Exam features:
  - Multiple choice questions
  - True/false questions
  - Short answer questions
  - Countdown timer with auto-submit
  - Real-time progress tracking
  - Automatic grading
  - Pass/fail status
- Docker Compose setup for easy deployment
- Comprehensive API documentation
- Full database schema with migrations

### Features
- Admin Dashboard with statistics
- Student Dashboard with exam history
- Real-time exam results
- Configurable passing scores
- Random question ordering option
- Detailed answer breakdown in results
